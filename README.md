# Sticker Board

Simple sticker board application



# Authors

# Build With

Easily could've been done with vanilla javascript
* OpenLayers - https://openlayers.org/en/latest/doc/

* Jquery - https://api.jquery.com/

* AngularJS - https://docs.angularjs.org/api


# Prerequisites
* Java 11
* Maven 3.6.2
* H2 DB 1.4

# How to Install
**Api**
* mvn spring-boot:run

**UI**
* npm install
* npm run serve
